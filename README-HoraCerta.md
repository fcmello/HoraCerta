# HoraCerta (Material Design 3 — verde #266e42)

Aplicativo simples para cálculo de jornada e banco de horas.

## Rodar local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build && npm run preview
```

## Deploy (Vercel)
- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
